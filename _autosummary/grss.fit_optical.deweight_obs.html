grss.fit_optical.deweight_obs

grss.fit_optical.deweight_obs(obs_array_optical, star_catalog_codes, observer_codes_optical, eff_obs_per_night=5, verbose=False)

Deweight observations that occur on the same night at the same observatory.

Parameters

  • obs_array_optical (array) – Optical observation data for the given body

  • star_catalog_codes (tuple) – Star catalog codes used for reducing each observation in obs_array_optical

  • observer_codes_optical (tuple) – Observer locations for each observation in obs_array_optical

  • eff_obs_per_night (int, optional) – Effective number of observations per night, by default 5

  • verbose (bool, optional) – Flag to print out information about the observations, by default False

Returns

  • obs_array_deweighted (array) – Deweighted Optical observation data for the given body

  • catalog_deweighted (tuple) – Star catalog codes used for reducing each observation in obs_array_deweighted

  • observer_loc_deweighted (tuple) – Observer locations for each observation in obs_array_deweighted
