grss.fit_utils.get_observer_info¶

grss.fit_utils.get_observer_info(observer_codes)¶

Creates a list of observer information for each observer code for passing into a grss propSimulation object

Parameters:

observer_codes (list) – Observer locations for each observation. Shape of each entry changes on type of observation: 1 (optical), 2 (radar), or 3 (doppler).

Returns:

observer_info – Information about each observer (Spice ID for parent body, longitude, geodetic latitude, and altitude). Shape of each entry changes on type of observation: 4 (optical), 8 (radar), or 9 (doppler).

Return type:

list
