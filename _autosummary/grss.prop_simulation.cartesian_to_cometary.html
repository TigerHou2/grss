grss.prop_simulation.cartesian_to_cometary

grss.prop_simulation.cartesian_to_cometary(epochMjd: float, cartesianState: List[float], GM: float = 0.00029591220828559115) → List[float]

Convert cartesian state to cometary state.

Parameters

  • epochMjd (real) – Epoch in modified Julian date.

  • cartesianState (list of real) – Cartesian state vector.

  • GM (real, optional) – Gravitational parameter of the central body, by default 0.00029591220828559115L.

Returns

cometaryState – Cometary state vector.

Return type

list of real
